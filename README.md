### Developer instructions
- compile C program
```
gcc -Wall filename.c -o filename
```

- lookup functions
```
man getpid
```

- find header files
```
ls /usr/include/signal.h
find /usr/include/ -name types.h
```

- check ubuntu version
```
lsb_release -a
```

- check linux version
```
uname -r
```

- adjust system limits
```
# as root
sudo bash
ulimit -a
```
